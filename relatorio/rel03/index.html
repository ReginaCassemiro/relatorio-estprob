3 Instalação do R e RStudio

Para utilizar o pacote LEEM, primeiro é necessário instalar a linguagem R, que será responsável pela execução dos comandos estatísticos. Em seguida, recomenda-se instalar o RStudio, ambiente gráfico que facilita a criação de scripts, organização dos arquivos e visualização dos resultados.

  • Download do R: https://cran.r-project.org/
  • Download do RStudio: https://posit.co/download/rstudio-desktop/

Após a instalação dos programas, basta abrir o RStudio para começar a utilizar os recursos da linguagem R.

4 Instalação do pacote LEEM

Após instalar o R e o RStudio, o próximo passo é adicionar o pacote LEEM ao ambiente de trabalho. O LEEM (Laboratório de Ensino de Estatística e Matemática) foi desenvolvido com finalidade educacional e reúne funções voltadas para o estudo da Estatística de maneira prática e simplificada.

Com esse pacote, é possível criar tabelas de frequência, gráficos estatísticos, calcular medidas descritivas, probabilidades e realizar testes estatísticos.

A instalação pode ser feita diretamente no RStudio com o comando:

install.packages("leem")

Após finalizar a instalação, é necessário carregar a biblioteca no ambiente:

  • Carregar pacote LEEM
library(leem)

Para verificar se o pacote está funcionando corretamente, pode-se utilizar:

  • Abrir interface do pacote
leem()

Com isso, o pacote estará pronto para uso nos próximos comandos estatísticos

5 Funções para tabulação de dados

A tabulação de dados é utilizada para organizar informações em tabelas, facilitando a interpretação dos resultados. No pacote LEEM, esse processo permite transformar dados brutos em classes com suas respectivas frequências.

Inicialmente, será criado um conjunto de dados aleatórios com distribuição normal. Em seguida, esses valores serão organizados em uma tabela de frequência.

# Gerar dados aleatórios
set.seed(10)
dados <- rnorm(1000, mean = 100, sd = 2)

# Transformar em objeto LEEM
base <- new_leem(dados)

# Criar tabela de frequência com 7 classes
tab <- tabfreq(base, k = 7)

# Exibir tabela
tab

Explicação dos comandos:

set.seed(10): fixa a sequência aleatória;

rnorm(): gera valores com distribuição normal;

new_leem(): transforma os dados em objeto compatível com o pacote;

tabfreq(): cria tabela de frequência;

k = 7: define o número de classes.

6 Apresentações gráficas

As apresentações gráficas são importantes para facilitar a visualização e interpretação dos dados. O pacote LEEM oferece recursos gráficos que auxiliam na análise estatística de forma prática e objetiva.

Entre os principais gráficos disponíveis destacam-se o histograma, a ogiva e o gráfico de pizza.

6.1 Histograma

O histograma permite visualizar a distribuição dos dados em classes, mostrando onde há maior concentração de valores.

tab |>
hist(
freq = "p",
bg = FALSE,
barcol = heat.colors(7),
xlab = "Variável X"
)

6.2 Ogiva

A ogiva representa as frequências acumuladas, sendo útil para análise de medianas e percentuais acumulados.

tab |>
ogive(
bars = TRUE,
both = TRUE,
barcol = c("blue","yellow"),
lpcol = "red",
pch = 8
)

6.3 Gráfico de Pizza

O gráfico de pizza mostra a proporção de cada classe em relação ao total de dados.

tab |>
piechart()

7 Medidas de posição e dispersão

As medidas de posição e dispersão são utilizadas para resumir informações importantes de um conjunto de dados. Enquanto as medidas de posição indicam valores centrais, as medidas de dispersão mostram o grau de variação dos dados.

7.1 Medidas de posição

No pacote LEEM, é possível calcular média, mediana e moda com o comando:

tab |>
mpos(grouped = FALSE)

Essas medidas ajudam a identificar o comportamento central dos dados analisados.

7.2 Medidas de dispersão

Para avaliar a variabilidade dos dados, utilizam-se variância, desvio padrão e coeficiente de variação.

7.3 Variância

tab |>
variance()

7.4 Desvio padrão

tab |>
sdev()

7.5 Coeficiente de variação

tab |>
cv()

Esses resultados permitem verificar se os dados estão mais concentrados ou mais dispersos em relação à média.

8 Distribuições de probabilidades

As distribuições de probabilidades são utilizadas para calcular a chance de ocorrência de determinados eventos aleatórios. No pacote LEEM, essas análises podem ser realizadas de forma simples por meio da função P().

Entre as distribuições mais utilizadas estão a normal e a binomial.

8.1 Distribuição Normal

A distribuição normal é bastante aplicada em estudos estatísticos, especialmente em fenômenos naturais e sociais.

O exemplo abaixo calcula a probabilidade de um valor estar entre -1 e 1.

P(-1 %<=X<=% 1, mean = 0, sd = 1)

8.2 Distribuição Binomial

A distribuição binomial é utilizada quando se deseja calcular a probabilidade de sucessos em um número fixo de tentativas.

No exemplo a seguir, calcula-se a probabilidade de obter exatamente 5 sucessos em 10 tentativas, com chance de 50% em cada uma.

P(5, dist = "binomial", size = 10, prob = 0.5)

Esses recursos mostram a utilidade do pacote LEEM em aplicações de probabilidade no ambiente R.

9 Intervalo de confiança

O intervalo de confiança é uma técnica estatística utilizada para estimar parâmetros populacionais com base em dados amostrais. Em vez de apresentar apenas um valor único, fornece-se um intervalo no qual existe determinada confiança de que o parâmetro verdadeiro esteja localizado.

Esse procedimento é muito utilizado para estimar médias, proporções e outros valores importantes em pesquisas e estudos estatísticos.

Esse recurso pode ser aplicado por meio de funções estatísticas disponíveis no R, como o teste t.

Exemplo de utilização:

dados <- c(10,12,11,9,13,10,12,11)
t.test(dados, conf.level = 0.95)

De modo geral, quanto maior o nível de confiança adotado, maior tende a ser a amplitude do intervalo estimado. Esse método auxilia na tomada de decisões com base em amostras.

10 Teste de hipóteses

O teste de hipóteses é um procedimento estatístico utilizado para verificar se uma afirmação sobre determinada população pode ser aceita ou rejeitada com base em uma amostra de dados.

Normalmente, trabalha-se com duas hipóteses:

  • Hipótese nula (H₀): representa a condição inicial a ser testada;
  • Hipótese alternativa (H₁): representa a possibilidade contrária.

Esse processo pode ser realizado no R por meio da função t.test(), utilizada para testes estatísticos de média.

Exemplo de utilização:

dados <- c(10,12,11,9,13,10,12,11)
t.test(dados, mu = 10)

Após a análise do valor-p ou da região crítica, toma-se a decisão estatística sobre a hipótese formulada.

Esse recurso é bastante importante em pesquisas científicas e estudos experimentais.
