1 📌 Introdução

A linguagem R é muito utilizada na área de estatística e análise de dados, sendo bastante importante dentro da Ciência de Dados. Com o aumento do uso de dados em diversas áreas, aprender R se tornou essencial para estudantes e profissionais que precisam analisar e interpretar informações.

Nesta aula prática, foi realizada uma introdução à linguagem R com base no curso R Básico 2024. Ao longo das aulas, foi possível conhecer desde os conceitos mais simples, como execução de comandos e criação de variáveis, até noções mais organizadas de estruturas de dados e uso de funções.

O curso apresenta o conteúdo de forma progressiva, começando pelo ambiente de trabalho, passando pela manipulação de dados e chegando até conceitos como funções, estruturas de controle e uso de pacotes. Isso facilita bastante o aprendizado, principalmente para quem está tendo o primeiro contato com programação.

A atividade permitiu entender melhor como o R funciona na prática e como ele pode ser utilizado no ambiente acadêmico para facilitar cálculos, organização de dados e análises.

2 🎯 Objetivos

Aprender os conceitos básicos da linguagem R a partir das aulas do curso R Básico.

  • Objetivos específicos
  • Conhecer o ambiente R e o RStudio
  • Executar comandos simples
  • Criar e manipular variáveis
  • Trabalhar com estruturas de dados básicas
  • Entender o uso de funções
  • Compreender a organização de códigos e scripts

3 📚 Fundamentação Teórica

O R é uma linguagem de programação voltada para análise estatística e manipulação de dados. Durante o curso, foi possível observar que o R funciona de forma baseada em objetos, ou seja, tudo que é criado pode ser armazenado e reutilizado posteriormente.

Inicialmente, são apresentados conceitos básicos, como a utilização do console e a execução de comandos simples. Em seguida, o curso aborda a criação de variáveis, operações matemáticas e organização de arquivos.

Outro ponto importante é o estudo das estruturas de dados, como vetores, matrizes, listas e data frames, que são fundamentais para organizar e analisar informações. Também são utilizadas funções, que permitem automatizar cálculos e facilitar o desenvolvimento dos códigos.

Além disso, são apresentadas estruturas de controle, como condições e repetições, que ajudam na tomada de decisões dentro do programa. O curso também destaca a importância da organização do código, incentivando o uso de boas práticas.

Por fim, é introduzido o uso de pacotes, que ampliam as funcionalidades do R, permitindo realizar análises mais completas. Dessa forma, o conteúdo estudado fornece uma base sólida para o aprendizado da linguagem.

4 ⚙️ Metodologia

A prática foi realizada no computador utilizando o RStudio.

Realizados os seguintes procedimentos:

  • Abertura do RStudio
  • Execução de comandos no console
  • Criação de variáveis
  • Realização de operações matemáticas
  • Manipulação de vetores e dados simples
  • Utilização de funções básicas
  • Observação dos resultados gerados

5 🔍 Resultados e Discussão

Com base nas aulas do curso, percebe-se que o R vai se tornando mais completo conforme o conteúdo avança, principalmente com o uso de estruturas mais organizadas, funções e pacotes.

Mesmo podendo se tornar mais complexo com o tempo, o R se mostra uma ferramenta muito útil e eficiente para análise de dados. No geral, os resultados foram positivos e contribuíram para um bom entendimento inicial da linguagem.

6 🧠 Considerações finais

O estudo da linguagem R, com base nas 26 aulas do curso R Básico 2024, permitiu compreender seus conceitos fundamentais e suas aplicações na análise de dados.

Ao longo da prática, foi possível observar que o R é uma ferramenta eficiente para realizar cálculos, organizar dados e aplicar conceitos estatísticos de forma simples. Os exemplos trabalhados ajudaram a entender melhor o funcionamento da linguagem na prática.

Como exemplo, foi possível utilizar comandos básicos para criação de variáveis e cálculos:

Código 
x <- 10
y <- 5
x + y
[1] 15

Também foi possível trabalhar com vetores e aplicar funções estatísticas:

Código 
valores <- c(2, 4, 6, 8, 10)
mean(valores)
[1] 6

Esses exemplos mostram como o R facilita a análise de dados mesmo em situações simples.

Os objetivos propostos foram alcançados, pois foi possível compreender o funcionamento básico da linguagem e aplicar comandos na prática.

Como continuidade, é importante realizar mais exercícios e aprofundar o uso de funções e estruturas mais avançadas, para desenvolver maior domínio da linguagem.
